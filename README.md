### **`temperMonkey脚本`** 快捷搜索


> 该脚本旨在使用一些常用网页时快捷定位到搜索框

### 使用方法

#### npmjs.com的搜索框定位

* mac：cmd + g
* windows： windows + g

#### www.bootcdn.cn的搜索框定位

* mac：cmd + g
* windows： windows + g
#### bilibili首页的搜索框定位

* mac：cmd + g
* windows： windows + g
#### bilibili搜索页的搜索框定位

* mac：cmd + g
* windows： windows + g
